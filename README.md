# Graph Neural Networks Blog

Welcome to my blog about Graph Neural Networks, PyTorch Geometric, and their applications in computational biology.

## Posts

- [Graph Convolutional Networks for Protein Function Prediction](./_posts/2025-11-01-gcn-protein-function.md) - A comprehensive tutorial on building GCNs with PyTorch Geometric for multi-label protein function classification.

## Setup

This is a Jekyll site hosted on GitHub Pages. To run locally:

```bash
bundle install
bundle exec jekyll serve
```

```
├── _config.yml          # Jekyll configuration

```
